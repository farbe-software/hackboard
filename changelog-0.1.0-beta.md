# Hackboard v0.1.0-beta

Initial release.

Hackboard is built on [Karabiner Elements](https://karabiner-elements.pqrs.org/) by Fumihiko Takayama. [Support his work](https://karabiner-elements.pqrs.org/docs/pricing/).

## What's in the box

- **Letter keys as modifiers** — Hold any letter key to activate a layer of shortcuts. Tap it normally and it types as usual, thanks to smart rollover that protects your typing at any speed.
- **Deep chaining** — Layer up to four levels deep. E.g. hold `A` → `C` → `M` → `K` to drill from "Apps" → "Chrome" → "Maps" → a specific location.
- **Independent left/right modifiers** — Right ⌘, ⌥, ⌃, and ⇧ can do different things than their left counterparts. E.g. hold Right ⌘ to type numbers from the home row.
- **Three action types** — Remap to keyboard shortcuts, insert text, or run shell commands (including app switching).
- **Visual editor** — Record key chains by performing them. No JSON editing needed.
- **Guided onboarding** — Interactive walkthrough with example keymaps to get started immediately.
- **Built on Karabiner Elements** — Ships with Hb-Karabiner-Elements (our fork, pinned to v15.0.0). Your existing Karabiner rules keep working alongside Hackboard keymaps.

## Requirements

- macOS 13 (Ventura) or later
- Apple Silicon recommended
